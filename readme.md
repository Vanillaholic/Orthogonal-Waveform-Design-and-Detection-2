# 多种探测波形仿真

- CW信号：

  - 采样频率：144kHz
  - 脉冲长度：6s
  - 占空比20%，wav总时长30s
  - 中心频率：3kHz
  - 调制率：0.05

- M序列
  - 采样频率：144kHz
  - 脉冲长度：6s
  - 占空比20%，wav总时长30s
  - 中心频率：3kHz
  - 以序列[1 0 0 1 1 0 1 0 1 1 1 1 0 0 0]对单个信号进行扩频
  - 调制率：0.05

- Costas信号：占空比50%

  - 采样频率：144kHz

  - 脉冲长度：6s

  - 占空比20%，wav总时长30s

  - 中心频率：3kHz  频率范围：2.5~3.5kHz

  - 调制率：0.05

  - 以矩阵对单个脉冲进行跳频(横轴为时间，纵轴为频率)

  - 公式：
    $$
    s(t) = A \cos \left( 2\pi \left( f_0 t + K \ln t \right) \right)
